About DiziQuote

DiziQuote is a mobile application built using Flutter that provides a collection of inspirational, motivational, and meaningful quotes.

Users can browse quotes, share them, and optionally sign in using Google to access personalized features.

Authentication

DiziQuote uses Google Sign-In through Firebase Authentication. Users sign in securely using their own Google accounts.
